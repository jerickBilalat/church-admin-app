## Get Started

1. **Install [Node 8 or higher](https://nodejs.org)**.
2. **Clone this repository.**
3. **Make sure you're in the directory you just created.**
4. **Install Node Packages.** - `npm install`
5. **Run the app.** - `npm run dev`
